# HxLauncher (零桌面 / 笨蛋桌面)

[English](#english) | [中文](#中文)

<a name="english"></a>
## English

**HxLauncher** (also known as "零桌面" or "笨蛋桌面") is a lightweight Android launcher application.

### Features

- 📱 **App Launcher** - Tap any icon to launch the corresponding application
- 🔍 **App Tracking** - Automatically tracks app installations and uninstallations
- 🌊 **Waterfall Layout** - Beautiful waterfall-style app display
- 🔄 **Auto Rotation** - Seamlessly switches between portrait and landscape modes based on gravity sensor
- 🔊 **Voice Control** - Support for voice commands to control the launcher
  - Say the app name to launch it directly
  - **Smart Learning**: When voice recognition fails, the system learns from user clicks and improves over time
  - **Privacy-Focused**: Minimal icons displayed, voice-only control for better privacy
  - **On-Demand Installation**: Auto-downloads apps that aren't installed yet
- 🌐 **Multi-language** - Supports internationalization for app names

## 中文

**HxLauncher** (零桌面 / 笨蛋桌面) 是一款轻量级 Android 桌面启动器应用。

### 功能特性

- 📱 **应用启动** - 点击图标即可启动对应应用
- 🔍 **应用跟踪** - 自动跟踪系统中应用的安装和卸载情况
- 🌊 **瀑布流布局** - 以美丽的瀑布流形式展示所有应用
- 🔄 **自动旋转** - 根据重力传感器在横屏与竖屏间快速切换
- 🔊 **语音控制** - 支持语音命令控制桌面
  - 喊出应用名字即可启动
  - **智能学习**: 识别失败时，系统会记住用户手动启动的应用关联，越用越聪明
  - **隐私保护**: 默认不显示应用图标，仅靠语音识别即可启动大部分应用，保护隐私
  - **按需安装**: 语音命中的应用未安装时，自动下载安装包
- 🌐 **国际化** - 支持应用名称国际化
